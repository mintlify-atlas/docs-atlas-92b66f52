## Overview

Protocol blocks enable communication using various network protocols beyond HTTP:

- **TCP/UDP**: Raw socket communication
- **FTP**: File transfer operations
- **SSH**: Remote command execution
- **IMAP**: Email retrieval via IMAP
- **SMTP**: Email sending
- **POP3**: Email retrieval via POP3

## TcpRequest

Send data over TCP sockets.

### Settings

<ParamField path="host" type="string" required>
  Target host (IP or domain)
</ParamField>

<ParamField path="port" type="number" default="80" required>
  TCP port number
</ParamField>

<ParamField path="data" type="string" required>
  Data to send (supports variable interpolation)
</ParamField>

<ParamField path="output_var" type="string" default="TCP_RESPONSE">
  Variable name to store response
</ParamField>

<ParamField path="timeout_ms" type="number" default="5000">
  Connection timeout in milliseconds
</ParamField>

<ParamField path="use_tls" type="boolean" default="false">
  Use TLS/SSL encryption
</ParamField>

<ParamField path="ssl_verify" type="boolean" default="true">
  Verify SSL certificates (when use_tls is true)
</ParamField>

<ParamField path="capture" type="boolean" default="false">
  Capture as user-visible variable
</ParamField>

### Example

```json
{
  "block_type": "TcpRequest",
  "label": "Custom Protocol",
  "settings": {
    "type": "TcpRequest",
    "host": "api.example.com",
    "port": 9000,
    "data": "HELLO <input.USER>\\r\\n",
    "use_tls": true
  }
}
```

## FtpRequest

Perform FTP file operations.

### Settings

<ParamField path="host" type="string" required>
  FTP server hostname
</ParamField>

<ParamField path="port" type="number" default="21" required>
  FTP port
</ParamField>

<ParamField path="username" type="string" required>
  FTP username
</ParamField>

<ParamField path="password" type="string" required>
  FTP password
</ParamField>

<ParamField path="command" type="string" default="LIST">
  FTP command:
  - `LIST` - List directory contents
  - `RETR` - Download file
  - `STOR` - Upload file
  - `DELE` - Delete file
  - `MKD` - Create directory
  - `RMD` - Remove directory
  - `CWD` - Change directory
</ParamField>

<ParamField path="remote_path" type="string">
  Remote file/directory path (for RETR, STOR, DELE, MKD, RMD, CWD)
</ParamField>

<ParamField path="local_path" type="string">
  Local file path for upload (STOR)
</ParamField>

<ParamField path="output_dir" type="string">
  Local directory for download (RETR)
</ParamField>

<ParamField path="output_var" type="string" default="FTP_RESPONSE">
  Variable name to store result
</ParamField>

<ParamField path="timeout_ms" type="number" default="10000">
  Operation timeout
</ParamField>

<ParamField path="capture" type="boolean" default="false">
  Capture as user-visible variable
</ParamField>

### Example

```json
{
  "block_type": "FtpRequest",
  "label": "List Files",
  "settings": {
    "type": "FtpRequest",
    "host": "ftp.example.com",
    "port": 21,
    "username": "<input.USER>",
    "password": "<input.PASS>",
    "command": "LIST",
    "remote_path": "/public"
  }
}
```

## SmtpRequest

Send emails via SMTP.

### Settings

<ParamField path="host" type="string" required>
  SMTP server hostname
</ParamField>

<ParamField path="port" type="number" default="587" required>
  SMTP port (587 for STARTTLS, 465 for SSL, 25 for plain)
</ParamField>

<ParamField path="username" type="string" required>
  SMTP username
</ParamField>

<ParamField path="password" type="string" required>
  SMTP password
</ParamField>

<ParamField path="use_tls" type="boolean" default="true">
  Use STARTTLS
</ParamField>

<ParamField path="ssl_verify" type="boolean" default="true">
  Verify SSL certificates
</ParamField>

<ParamField path="command" type="string" default="EHLO">
  SMTP command to send
</ParamField>

<ParamField path="action" type="string" default="VERIFY">
  Action type:
  - `VERIFY` - Just test login
  - `SEND_EMAIL` - Send actual email
</ParamField>

<ParamField path="from" type="string">
  Sender email (defaults to username)
</ParamField>

<ParamField path="to" type="string">
  Recipient emails (comma-separated)
</ParamField>

<ParamField path="subject" type="string">
  Email subject
</ParamField>

<ParamField path="body_template" type="string">
  Email body (supports variable interpolation)
</ParamField>

<ParamField path="output_var" type="string" default="SMTP_RESPONSE">
  Variable name to store result
</ParamField>

<ParamField path="timeout_ms" type="number" default="10000">
  Operation timeout
</ParamField>

<ParamField path="capture" type="boolean" default="false">
  Capture as user-visible variable
</ParamField>

### Example

```json
{
  "block_type": "SmtpRequest",
  "label": "Send Email",
  "settings": {
    "type": "SmtpRequest",
    "host": "smtp.gmail.com",
    "port": 587,
    "username": "sender@example.com",
    "password": "<data.APP_PASSWORD>",
    "action": "SEND_EMAIL",
    "to": "recipient@example.com",
    "subject": "Test Email",
    "body_template": "Hello from IronBullet!"
  }
}
```
