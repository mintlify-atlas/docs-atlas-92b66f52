The `Pipeline` struct represents a complete automation config with blocks, data sources, proxy settings, and browser configuration.

## DataSettings

Defines how to parse input data (wordlist).

<ParamField path="wordlist_type" type="string" required>
  Preset format name: `"Credentials"`, `"Url"`, `"Custom"`, etc.
</ParamField>

<ParamField path="separator" type="char" required>
  Character to split each line on (typically `:` for credentials)
  
  **Default:** `':'`
</ParamField>

<ParamField path="slices" type="Vec<string>" required>
  Ordered variable names for split parts
  
  **Example:** `["USER", "PASS"]` for credentials
</ParamField>

## ProxySettings

Proxy rotation and health checking configuration.

<ParamField path="proxy_mode" type="ProxyMode" required>
  Rotation mode: `None`, `Rotate`, `Sticky`, `CpmLimited`
</ParamField>

<ParamField path="proxy_sources" type="Vec<ProxySource>" required>
  Proxy sources (file, URL, inline list)
</ParamField>

<ParamField path="ban_duration_secs" type="u64" required>
  How long to ban a proxy after detection (seconds)
  
  **Default:** `300`
</ParamField>

<ParamField path="max_retries_before_ban" type="u32" required>
  Failed attempts before banning a proxy
  
  **Default:** `3`
</ParamField>

<ParamField path="cpm_per_proxy" type="u32">
  Max checks per minute per proxy (for `CpmLimited` mode)
</ParamField>

<ParamField path="proxy_groups" type="Vec<ProxyGroup>">
  Named proxy groups for different pools
</ParamField>

<ParamField path="active_group" type="string">
  Active proxy group name (empty = use default sources)
</ParamField>

## BrowserSettings

Browser and TLS emulation configuration.

<ParamField path="browser" type="string" required>
  Browser type: `"chrome"`, `"firefox"`, `"edge"`
  
  **Default:** `"chrome"`
</ParamField>

<ParamField path="ja3" type="string?">
  JA3 TLS fingerprint override (hex string)
</ParamField>

<ParamField path="http2_fingerprint" type="string?">
  HTTP/2 SETTINGS frame fingerprint
</ParamField>

<ParamField path="user_agent" type="string?">
  Custom User-Agent override
</ParamField>

## RunnerSettings

Job execution limits and timeout configuration.

<ParamField path="threads" type="u32">
  Number of concurrent worker threads
  
  **Default:** `1`
</ParamField>

<ParamField path="skip" type="u32">
  Skip the first N data lines
  
  **Default:** `0`
</ParamField>

<ParamField path="take" type="u32">
  Maximum data lines to process (0 = all)
  
  **Default:** `0`
</ParamField>

<ParamField path="continue_statuses" type="Vec&lt;BotStatus&gt;">
  Statuses that re-queue the data line (e.g., `["Retry"]`)
  
  **Default:** `[]`
</ParamField>

<ParamField path="custom_status_name" type="string">
  Custom label for Custom result status
  
  **Default:** `""`
</ParamField>

<ParamField path="max_retries" type="u32">
  Max retry attempts per data line before marking as error
  
  **Default:** `0`
</ParamField>

<ParamField path="concurrent_per_proxy" type="u32">
  Maximum concurrent uses per proxy (0 = unlimited)
  
  **Default:** `0`
</ParamField>

<ParamField path="start_threads_gradually" type="bool">
  Start threads gradually instead of all at once
  
  **Default:** `false`
</ParamField>

<ParamField path="gradual_delay_ms" type="u32">
  Delay between starting each thread (milliseconds)
  
  **Default:** `1000`
</ParamField>

## OutputSettings

Result output configuration.

<ParamField path="save_to_file" type="bool">
  Save hits to text files
  
  **Default:** `true`
</ParamField>

<ParamField path="save_to_database" type="bool">
  Save hits to SQLite database
  
  **Default:** `false`
</ParamField>

<ParamField path="include_response" type="bool">
  Include full response body in output (debug mode)
  
  **Default:** `false`
</ParamField>

<ParamField path="output_directory" type="string">
  Output directory for result files
  
  **Default:** `"results"`
</ParamField>

<ParamField path="output_format" type="string">
  Custom output format template
  
  **Default:** `"{data} | {captures}"`
</ParamField>

<ParamField path="database_path" type="string">
  SQLite database file path
  
  **Default:** `""`
</ParamField>

<ParamField path="output_format_type" type="OutputFormat">
  Output file format: `Txt`, `Csv`, `Json`
  
  **Default:** `Txt`
</ParamField>

<ParamField path="capture_filters" type="Vec&lt;CaptureFilter&gt;">
  Filters applied before writing output
  
  **Default:** `[]`
</ParamField>
